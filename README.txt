Um das Projekt laufen zu können müssen:
 - Docker installiert selenium
 - Selenium-Webscraper muss installiert selenium (Achtung - Richtige Version für Chrome muss ausgewählt sein und unter Pfad )
 - requirements.txt muss mittels "pip install -r requirements.txt" installiert werden


Wir verwenden die gleichen Python packages, damit es zu keinen Missverständnissen führt.

Falls weitere installiert werden müssen, können diese im 'requirements.txt' hinzugefügt werden.
